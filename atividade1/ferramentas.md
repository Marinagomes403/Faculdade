# HTML : quais elementos estão sendo usados?
 head: para adicionar o cabeçalho da página;
 body: para fazer o corpo do texto;
 H1,H2,H3 e H5: Para definir os títulos;
 Nav: é usada para definir uma seção de navegação principal em uma página;
 ul e li: Para fazer uma lista não ordenada;
 br: Para pular linha, e deixar o site orgânizado;
 form: para adicionar um formulário;
 fieldset: para agrupar campos relacionados dentro de um formulário;
 legend: para definir uma legenda ou título para grupo de campos de um formulário;
 link: para direcionar links;
 div: para agrupar outros elementos, servindo como uma divisão na página;
 label: para colocar um texto de rótulo no campo de formulário, como um campo de entrada;
 input: para criar campos interativos nos formulários;
 p: Para fazer um parágrafo;
 a link: Para direcionar links externos;
 img: Para adicionar uma imagem; 

# css : quais propriedades estão sendo usadas?
 Background-color: Para adicionar uma cor ao fundo do elemento;
 Color: Adicionar cor ao elemento;
 Font-size: Para definir o tamanho da fonte;
 Font-family: para definir o estilo da fonte;
 Margin: para adicionar margem;
 Padding: para criar um espaço entre seu conteúdo e sua borda;
 Box-sizing: para controlar como a largura e a altura de um elemento são calculadas;
 display: para definir e controlar o comportamento do elemento no site;
 Gap: para definir o espaço entre os itens;
 List-style: para personalizar a aparência dos marcadores das listas;
 Possition: para definir a posição do elemento;
 Left: é para definir a posição da esquerda a direita de um elemento;
 Transform: Para transformar o elemento;
 Font-weight: para o peso de um caractere no texto, deixando mais claro ou mais escuro;
 Font-style: para estilizar a fonte;
